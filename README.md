# Convert API 

## How to Install and run
- `git clone https://github.com/Eirfand1/convert-api` 
- `cd convert-api`
- `npm install`
- `npm run start:build`

### POST `/convert/docx-to-pdf`

header: form-data


| key  |  body     | tipe       |
|:-----|:---------:|:----------:|
| file | `file.docx` | file(docx) |
